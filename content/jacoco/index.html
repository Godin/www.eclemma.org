Jacoco Java Code Coverage Library

JaCoCo is a new code coverage library for Java, which is currently developed based on the lessons learned from using and integration existing libraries over the last three years.

Status

A working prototype is available to evaluate implementation strategies and verify the basic ideas. Please refer to the latest documentation for details. While the prototype already provides Ant tasks for coverage analysis, this is not a stable library or fully usable tool yet. Please take a look at the latest version:

  • Example Coverage Report
  • Documentation
  • Download

The source code is available from the Subversion repository at https://eclemma.svn.sourceforge.net/svnroot/eclemma/jacoco/trunk (all sub-folders are Eclipse projects).
