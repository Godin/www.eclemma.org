Download and Installation

EclEmma ships as a small set of Eclipse plug-ins. The overall size of the three plug-ins is around 700 Kbyte.

Prerequisites

EclEmma requires Eclipse 3.1 or higher and has no dependencies on a particular operating system. Of course your Eclipse installation needs to contain the Java development tools (JDT) which is included in the default SDK installation.

Update Site

The update site for EclEmma is http://update.eclemma.org/. Perform the following steps to install EclEmma from the update site:

1. From your Eclipse menu select Help → Software Updates → Find and Install...

2. Check Search for new features and press Next.

3. Press New Remote Site... and enter http://update.eclemma.org/ for the update site URL:

4. Press Finish.

5. Follow the steps in the installation wizard.

Manual Installation

For manual installation please download the latest EclEmma release from the SourceForge.net file release system. The downloaded archive contains these files:

  • plugins/com.mountainminds.eclemma.feature_x.y.z.jar - EclEmma feature definition
  • plugins/com.mountainminds.eclemma.core_x.y.z.jar    - Core implementation
  • plugins/com.mountainminds.eclemma.ui_x.y.z.jar      - User interface
  • plugins/com.mountainminds.eclemma.doc_x.y.z.jar     - Online help

Simply unzip the archive into your Eclipse installation and restart Eclipse.
