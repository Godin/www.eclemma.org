Download and Installation

EclEmma shippes as a small set of Eclipse plug-ins. The overall size if the three plaugins is around 700 Kbyte.

Prerequisites

EclEmma requires Eclipse 3.1 or higher and has no dependencies on a particuar operating system. Of course your Eclipse installation needs to contain the Java development tools (JDT) whitch is included in the default SDK installation.

Download

The latest version along with release notes is available at the SourceForge.net file release system.

Installation

The downloaded archive contains three plug-ins:

  • plugins/com.mountainminds.eclemma.core_x.y.z.jar - Core implementation (required)
  • plugins/com.mountainminds.eclemma.ui_x.y.z.jar   - User interface (required)
  • plugins/com.mountainminds.eclemma.doc_x.y.z.jar  - Online help (optional)

Simply unzip the archive into your Eclipse installation and restart Eclipse. The installation was successful if you can see the coverage launcher in the toolbar of the Java perspective:
