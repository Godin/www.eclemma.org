Download and Installation

The latest version along with release notes is now available at SourceForge.net.

Installation

EclEmma requires Eclipse 3.1 or higher and has no dependencies on a particuar platform. The downloaded archive contains three plug-ins:

  • plugins/com.mountainminds.eclemma.core_x.y.z.jar - Core implementation (required)
  • plugins/com.mountainminds.eclemma.ui_x.y.z.jar   - User interface (required)
  • plugins/com.mountainminds.eclemma.doc_x.y.z.jar  - Online help (optional)

Simply unzip the archive into your Eclipse installation and restart Eclipse. See the chapter "EclEmma Java Code Coverage" in the online help system for details.
