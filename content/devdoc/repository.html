Source Repository

The source base of EclEmma is maintained in a Subversion repository at SourceForge. The repository's URL is

https://svn.sourceforge.net/svnroot/eclemma

You might also browse the Subversion repository online.

Repository Organisation

The latest versions of the plug-ins are available at

https://svn.sourceforge.net/svnroot/eclemma/eclemma/trunk

For each release there is a tagged version at

https://svn.sourceforge.net/svnroot/eclemma/eclemma/tags

The content of this website is also managed via Subversion:

https://svn.sourceforge.net/svnroot/eclemma/eclemmasite
