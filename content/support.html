Support

The EclEmma plug-ins ship with online help, see chapter EclEmma Java Code Coverage in the Eclipse help system after you installed EclEmma. The latest user documentation is also available online on this site.

User Support

If the documentation does not answer your question please post your issue in the help forum.

Bugs and Feature Requests

Always make sure you have installed the latest version. If you encountered a bug in the latest EclEmma version please report it to the bug tracker. There is a separate tracker for feature requests.
